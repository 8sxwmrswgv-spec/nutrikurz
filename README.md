# NutriKurz secure verzia

## Štruktúra

public/index.html = verejná stránka
private-videos/video1.mp4 = súkromné videá
server.js = backend, platby, emaily, kódy, ochrana videí
.env = tajné nastavenia

## Inštalácia

npm install

## Spustenie

npm start

Potom otvor:

http://localhost:3000

## Videá

Vlož videá do priečinka private-videos:

video1.mp4
video2.mp4
...
video13.mp4

Nedávaj ich do public ani vedľa HTML súboru.
